**直接回答：零基礎學 AI 的正確起點，不是背一堆 Prompt 技巧，而是安裝「桌面版 AI Agent」（Claude Code 或 Codex），取得工具權限，然後動手做出第一個真實作品。** 我推薦的第一個作品是「社群自動發文系統」，因為它會逼你把 AI 自動化的所有基本功走過一遍。

## 為什麼不是先學 Prompt 技巧？

網路上的 Prompt 教學多半停留在「怎麼問問題」，但 AI 真正的生產力來自**讓它執行任務**：操作檔案、串接平台、跑自動化流程。這些只有桌面版 Agent 做得到，[網頁版和免費 AI 做不到](/blog/why-desktop-claude-code-codex/)。

## 我建議的四步學習路線

這也是我的課程大綱，自學者可以照著走：

1. **裝好桌面版 Claude Code / Codex**：安裝、權限設定、環境變數，並訂閱每月 20 美元的方案即可（[Claude 和 Codex 怎麼選](/blog/claude-or-codex-budget/)）
2. **申請 Meta API**：[三大 API 學一個就通](/blog/meta-api-application/)，重點是 FB 帳號與開發者後台的綁定
3. **跑通自動發文**：[權限設定與實測](/blog/auto-posting-permissions/)，避開被 Meta 判定異常的坑
4. **把流程寫成 Skill**：[讓 AI 記住你的工作流](/blog/what-is-ai-skill/)，從「玩 AI」升級成「用 AI 工作」

走完這四步，你就擁有 AI 自動化的完整地基。之後想做 AI 客服、AI 自動拍片、AI 爬蟲，都是同一套邏輯的延伸。
