**直接回答：Claude Code 是 Anthropic 推出的「桌面版 AI Agent」——和網頁版聊天不同，它能實際操作你的電腦：讀寫檔案、設定環境變數、填入 API Key、執行自動發文腳本。** Codex 則是 OpenAI（ChatGPT 訂閱附帶）的同類工具。想做 AI 自動化（AI 客服、自動拍片、自動發文、爬蟲），第一步永遠是安裝這兩套其中之一，讓 AI 從「聊天對象」變成「能執行任務的 Agent」。

## AI Agent 是什麼？

一句話：**能代替你「動手做事」的 AI**。網頁版 ChatGPT / Claude 只能「回答」，你還是得自己動手；AI Agent 拿到工具權限後，可以自己安裝套件、串接 API、跑程式、驗證結果——你負責用中文下指令和驗收。Claude Code、Codex、OpenClaw（龍蝦）、Hermes（愛馬仕）都屬於這一類。

## 網頁版和桌面版差在哪？

| 能力 | 網頁版 Chat | 桌面版 Claude Code / Codex |
| --- | --- | --- |
| 回答問題 | ✅ | ✅ |
| 讀寫你電腦上的檔案 | ❌ | ✅ |
| 設定環境變數、填入 API Key | ❌ | ✅ |
| 執行自動發文腳本 | ❌ | ✅ |
| 記住你的工作流（Skill） | ❌ | ✅ |

網頁版每次對話都是從零開始，而桌面版 Agent 可以真正在你的電腦上安裝套件、串接第三方平台、把常用流程存成可重複執行的 Skill。

## 為什麼不用 n8n 或其他自動化平台？

n8n 這類流程平台適合已經知道「每個節點要做什麼」的人，新手光是理解節點與憑證設定就會卡關。桌面版 AI Agent 的優勢是：**你用中文描述需求，AI 幫你把技術細節做完**。等你透過 Agent 理解了整個流程，之後要不要搬去 n8n 都很容易。

## 免費 AI 為什麼不推薦？

免費模型的穩定度和能力通常明顯落後，同一件事會反覆出錯。你花掉的除錯時間，換算下來遠超過每月 20 美元的訂閱費。要把免費 AI 用出生產力，通常需要很熟 Prompt 工程的資深工程師——那不是新手該走的路。

## 新手的起步清單

1. 訂閱 ChatGPT 或 Claude 的月費方案（20 美元擇一即可，[預算只夠選一個該選哪個？](/blog/claude-or-codex-budget/)）
2. 安裝桌面版 Claude Code 或 Codex
3. 完成工具權限與環境設定（我的課程 Module 01 就是專門處理這一步）

這是後續一切 AI 自動化的地基：無論你之後要做 AI 客服、AI 自動拍片還是 AI 爬蟲，都是從這裡開始。
