**直接回答：可以。不會寫程式、文組出身都不是問題——AI Agent 已經把原本需要工程師的技術門檻，優化成「用中文對話」就能完成的工作流。** 只要你會操作電腦，就能從零完成桌面版 Claude Code / Codex 的安裝、Meta API 串接，做出自己的自動發文系統。我的學員裡，老師、行銷、個人品牌經營者這類非技術背景的人，反而是多數。

## 「不會寫程式」在 AI 時代還是障礙嗎？

過去要做社群自動化，你得會寫程式、讀 API 文件、處理伺服器。現在流程變成：

1. 你用中文告訴 AI Agent 想做什麼（「幫我每天早上 9 點發一篇貼文到 Threads」）
2. AI 寫程式、填設定、跑測試
3. 你確認結果，不滿意就再用中文修改

你的角色從「工程師」變成「出一張嘴的主管」，需要學的是**怎麼下對指令、怎麼驗收**，而不是語法。

## 哪些人最適合這樣學？

- **老師與講師**：提升備課效率、增加互動設計、把 AI 用進實際教學
- **個人品牌經營者**：需要大量產出內容、社群素材與銷售頁
- **想系統學 AI 的實作者**：不想再停留在零散技巧，想建立完整方法與實戰作品

## 新手最容易卡關的地方

依我一對一教學的經驗，零基礎學員的卡點幾乎都不是「程式」，而是：

- 工具權限與環境設定（Module 01 處理）
- Meta 開發者後台的申請流程與帳號綁定（Module 02 處理）
- 發文權限設定錯誤導致被 Meta 判定異常（Module 03 處理）
