**直接回答：預算只夠選一個的話，我推薦 Codex。** 原因是 ChatGPT 5.5 更新後，Codex 的使用額度比 Claude 同價位方案多很多；而且 Codex 的額度也可以拿來當 OpenClaw（龍蝦）或 Hermes 這類工具的「飼料」，相較之下比較不用擔心被封號。

## 兩者的定位差異

- **Claude Code**：Agent 能力與程式品質常被認為略勝一籌，生態系（Skill、MCP）成熟，我自己的主力工具。
- **Codex（ChatGPT 訂閱附帶）**：5.5 之後額度給得非常大方，同樣 20 美元能跑的任務量多很多，對新手練習特別友善。

新手階段最大的成本其實是「反覆試錯的額度消耗」，所以額度多寡比模型些微的能力差距更重要——這是我推薦 Codex 當第一個訂閱的主因。

## 什麼情況下反而該選 Claude？

- 你身邊的教學資源（老師、社群、課程）以 Claude Code 為主，跟著用可以少走路
- 你的工作流已經大量依賴 Claude 的 Skill / MCP 生態

如果你是跟我上一對一課程，兩套我都能教，開課前會依你的預算和目標幫你選定一套主力工具。
