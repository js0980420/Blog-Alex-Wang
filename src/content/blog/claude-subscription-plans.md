**直接回答：新手訂 Claude Pro（月付 20 美元、年付約 17 美元）就夠了，裡面已包含 Claude Code。** Max 是給重度使用者的：100 美元／月是 Pro 的 5 倍額度、200 美元／月是 20 倍，先用 Pro 撞到額度上限再考慮升級也不遲。

## Claude 訂閱方案總覽

| 方案 | 費用（美元／月） | 內容 |
| --- | --- | --- |
| Free | 0 | 只有網頁聊天，**沒有** Claude Code |
| Pro | 20（年付約 17） | 含 Claude Code，新手起步的標準選擇 |
| Max 5x | 100 | Pro 的 5 倍額度、高流量時段優先權 |
| Max 20x | 200 | Pro 的 20 倍額度、搶先體驗新功能 |

（價格以官網為準，方案內容可能調整。）

## 新手怎麼選？

1. **想學 AI 自動化 → 至少 Pro。** 免費版沒有 Agent 工具權限，[做不了自動化](/blog/why-desktop-claude-code-codex/)。
2. **預算只有一份 20 美元** → 先看 [ChatGPT 跟 Claude 的比較](/blog/claude-or-codex-budget/)，我目前推薦從 ChatGPT Plus（附 Codex）開始，額度比較耐用。
3. **已經確定主力用 Claude Code** → Pro 起步，搭配省額度技巧（課程 Module 01 會教），新手階段一個月 20 美元的額度就夠開始了。

## 額度不夠的訊號

- 每天都收到「額度已用完，請等待重置」
- 跑長任務（如自動拍片、大量發文）常在中途被打斷

出現這兩種情況再升 Max 5x 就好——直接跳 200 美元的 20x 對新手來說幾乎一定是浪費。

## 省額度的小技巧

- 把常用流程寫成 [Skill](/blog/what-is-ai-skill/)，避免每次重新解釋浪費 token
- 日常任務改用便宜的模型跑，例如 Claude 的 Haiku 或 GPT 5.5，把高階模型的額度留給開發和除錯
- 一次把需求講清楚，比來回追問省很多額度
